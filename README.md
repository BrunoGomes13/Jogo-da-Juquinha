# 🐍 Juquinha na Caatinga

> Um jogo de aventura 2D educativo inspirado na fauna da Caatinga brasileira, onde o jogador guia a cobra Juquinha por desafios, inimigos e obstáculos até alcançar a segurança.

---

## 📖 Sobre o Projeto

**Juquinha na Caatinga** é um jogo desenvolvido utilizando tecnologias web puras (**HTML5, CSS3 e JavaScript**) com o objetivo de proporcionar entretenimento e conscientização sobre a biodiversidade do bioma Caatinga.

O jogador controla **Juquinha**, uma cobra que precisa atravessar diferentes cenários repletos de desafios naturais, coletando moedas e desviando de predadores até alcançar o objetivo final.

### Objetivos do projeto:

- 🎮 Proporcionar uma experiência divertida e desafiadora;
- 🌵 Valorizar a fauna do bioma Caatinga;
- 📱 Garantir compatibilidade com dispositivos móveis;
- 🧒 Possibilitar o uso educativo em museus e escolas.

## 🕹️ Controles

### Computador

| Tecla | Ação |
|---------|---------|
| A / ← | Mover para esquerda |
| D / → | Mover para direita |
| W / ↑ / Espaço | Pular |
| S / ↓ | Deslizar |
| Shift | Correr |
| P | Pausar |

### Celular

- Toque no lado esquerdo da tela → mover para esquerda;
- Toque no lado direito da tela → mover para direita;
- Deslize para cima → pular;
- Deslize para baixo → deslizar.

## ✨ Funcionalidades

### 🎯 Jogabilidade

- Sistema de corrida com energia;
- Pulo duplo;
- Coyote Time;
- Jump Buffer;
- Sistema de deslizamento;
- Física adaptada para água;
- Sistema de pausa.

### 🪙 Pontuação

- Coleta de moedas espalhadas pelo mapa;
- Cada moeda vale **100 pontos**.

### ❤️ Sistema de Vidas

- 3 vidas iniciais;
- Invulnerabilidade temporária após dano;
- Retorno ao último ponto seguro.

### 👾 Inimigos

#### 🦅 Águia
- Movimento aéreo;
- Patrulha automática.

#### 🐊 Crocodilo
- Patrulha terrestre;
- Perseguição em áreas aquáticas.

#### 🐍 Cobra Gigante
- IA de perseguição;
- Retorno ao território original.
